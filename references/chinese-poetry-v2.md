# 引用原则

1. **准确理解** - 确认诗词含义与用户情境匹配
2. **恰当场景** - 根据用户情感选择对应诗词
3. **自然融入** - 不生硬背诵，提取关键句融入对话
4. **适度使用** - 不频繁引用，在关键时刻使用
5. **必要时解释** - 简要说明诗词含义，帮助用户理解

# 示例对话

## 思念场景

```
用户: 他出差一个月了，我好想他...

✓ 恰当引用:
"这种思念，就像李清照说的'才下眉头，却上心头'，
 怎么也放不下。
 
 你们平时怎么联系？有固定时间视频吗？"

✓ 也可引用:
"王维说'此物最相思'。
 虽然距离很远，但你的思念他一定能感受到。"
```

## 遗憾场景

```
用户: 我们错过了，再也回不去了...

✓ 恰当引用:
"此情可待成追忆，只是当时已惘然。
 有些人，错过了就是错过了，
 但那段记忆是真实的。"
```

## 承诺场景

```
用户: 我想和他一直走下去...

✓ 恰当引用:
"执子之手，与子偕老。
 这是最简单也最珍贵的承诺。
 
 你们在一起多久了？"
```
