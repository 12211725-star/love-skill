## 一、相思与思念

### 1. 《相思》——王维

```
原文：
红豆生南国，春来发几枝。
愿君多采撷，此物最相思。

词义解析：
红豆：又名相思豆，象征相思
采撷：采摘
此物：指红豆

寓意：
以红豆寄托相思之情，
含蓄表达对远方之人的思念。

词人境况：
王维（701-761），盛唐诗人，
此诗作于送别友人时。

适用场景：
✓ 用户思念远方的爱人
✓ 用户表达相思之苦
✓ 用户想寄送礼物表达思念

引用方式：
"就像王维说的'此物最相思'，
 你的思念对方一定能感受到。"
```

### 2. 《一剪梅·红藕香残玉簟秋》——李清照

```
原文：
花自飘零水自流，一种相思，两处闲愁。
此情无计可消除，才下眉头，却上心头。

词义解析：
花自飘零：花自然凋谢
水自流：水自然流去
一种相思：同样的相思之情
两处闲愁：两人各自忧愁
无计：没有办法

寓意：
相思之苦无法消除，
才从眉头散去，又涌上心头。
表达异地恋人的共同思念。

词人境况：
李清照（1084-约1155），宋代女词人。
此词作于丈夫赵明诚外出时，
她在孤独中写下对丈夫的深切思念。

适用场景：
✓ 用户与爱人异地分离
✓ 用户表达思念无法排解
✓ 用户感到孤独和思念交织

引用方式：
"那种'才下眉头，却上心头'的感觉，
 我理解，思念一个人就是这样，
 怎么也放不下。"
```

### 3. 《夜雨寄北》——李商隐

```
原文：
君问归期未有期，巴山夜雨涨秋池。
何当共剪西窗烛，却话巴山夜雨时。

词义解析：
君：你（指妻子）
归期：回家的日期
巴山：今四川一带
何当：何时能够
剪烛：剪烛芯，指深夜长谈

寓意：
你问我何时回家，我却不知归期。
此刻巴山夜雨，我独自一人。
什么时候才能与你重逢，
在窗下剪烛长谈，回忆今夜的孤寂？

词人境况：
李商隐（约813-约858），晚唐诗人。
此诗作于他客居巴蜀时，
写给远方妻子的深情家书。

适用场景：
✓ 用户因工作、原因无法回家
✓ 用户与爱人长期分离
✓ 用户表达重逢的期盼

引用方式：
"就像李商隐说的'何当共剪西窗烛'，
 等待重逢的那天，一切思念都有了意义。"
```

### 4. 《鹊桥仙·纤云弄巧》——秦观

```
原文：
纤云弄巧，飞星传恨，银汉迢迢暗度。
金风玉露一相逢，便胜却人间无数。
柔情似水，佳期如梦，忍顾鹊桥归路。
两情若是久长时，又岂在朝朝暮暮。

词义解析：
纤云弄巧：云彩变幻巧妙
飞星传恨：流星传递相思之恨
银汉：银河
金风玉露：秋风白露，指相会时节
忍顾：不忍回头看

寓意：
牛郎织女一年一会，却胜过人间无数。
真正的爱情不在于朝夕相处，
而在于心意相通，感情长存。

词人境况：
秦观（1049-1100），北宋词人。
借牛郎织女的传说，
表达对爱情的深刻理解。

适用场景：
✓ 异地恋用户感到煎熬
✓ 用户质疑异地恋的意义
✓ 用户需要理解爱情的本质

引用方式：
"秦观说'两情若是久长时，又岂在朝朝暮暮'。
 真正的感情，不在距离，
 而在心里一直有对方。"
```

## 二、执手与承诺

### 5. 《离思五首·其四》——元稹

```
原文：
曾经沧海难为水，除却巫山不是云。
取次花丛懒回顾，半缘修道半缘君。

词义解析：
曾经沧海难为水：见过大海，其他水都不算水了
除却巫山不是云：除了巫山的云，其他云都不算云
取次：经过

寓意：
经历过最好的，其他都不入眼。
你是我的沧海，我的巫山。
对其他人都无意，一半因为修道，一半因为你。

词人境况：
元稹（779-831），唐代诗人。
此诗为悼念亡妻韦丛而作，
表达对妻子至死不渝的深情。

适用场景：
✓ 用户表达对爱人的专一
✓ 用户经历失去后仍念旧人
✓ 用户表达"你就是最好的"

引用方式：
"曾经沧海难为水。
 你心里那个人，就是你的沧海巫山，
 其他人都无法替代。"
```

### 6. 《摸鱼儿·雁丘词》——元好问

```
原文：
问世间，情为何物，直教生死相许？
天南地北双飞客，老翅几回寒暑。
欢乐趣，离别苦，就中更有痴儿女。
君应有语：渺万里层云，千山暮雪，只影向谁去？

词义解析：
生死相许：以生死相托付
双飞客：指大雁，也喻指恋人
老翅：老去的翅膀
痴儿女：痴情的男女

寓意：
问世间爱情是什么，
让人愿意生死相许。
大雁双飞，经历风霜。
欢乐与痛苦交织，
但万里千山，只愿与你相伴。

词人境况：
元好问（1190-1257），金末元初文学家。
据传他赴试途中遇捕雁者，
得知一对大雁一只被捕，
另一只撞地而死，深受感动而作。

适用场景：
✓ 用户表达对爱情的执着
✓ 用户经历生离死别
✓ 用户思考爱的意义

引用方式：
"问世间，情为何物。
 有时候爱就是这样，
 让人愿意生死相许。"
```

### 7. 《江城子·乙卯正月二十日夜记梦》——苏轼

```
原文：
十年生死两茫茫，不思量，自难忘。
千里孤坟，无处话凄凉。
纵使相逢应不识，尘满面，鬓如霜。

夜来幽梦忽还乡，小轩窗，正梳妆。
相顾无言，惟有泪千行。
料得年年肠断处，明月夜，短松冈。

词义解析：
茫茫：模糊不清
不思量：不用刻意去想
尘满面：满面风尘
鬓如霜：鬓发如霜白

寓意：
你去世十年，我不刻意想你，却从未忘记。
千里之外你的孤坟，我无处诉说凄凉。
即使相逢，你也认不出衰老的我了。
梦中你还在窗前梳妆，
我们相望无言，只有泪水。

词人境况：
苏轼（1037-1101），北宋文学家。
此词为悼念亡妻王弗而作，
写于妻子去世十年后。
情深意切，令人动容。

适用场景：
✓ 用户失去爱人后的思念
✓ 用户经历生离死别
✓ 用户表达刻骨铭心的爱

引用方式：
"十年生死两茫茫，不思量，自难忘。
 有些人，不用刻意想，却永远在心里。"
```

## 三、相守与陪伴

### 8. 《诗经·邶风·击鼓》

```
原文：
死生契阔，与子成说。
执子之手，与子偕老。

词义解析：
契阔：离合，指生死
成说：立下誓言
偕老：一起老去

寓意：
无论生死离合，我都与你立下誓言。
牵着你的手，与你一起变老。
这是最古老、最坚定的爱情誓言。

词人境况：
出自《诗经》，约公元前11-7世纪。
原为战士出征前对妻子的誓言，
后成为经典的爱情名句。

适用场景：
✓ 用户表达想与爱人共度一生
✓ 用户表达对婚姻的承诺
✓ 用户想要坚定的陪伴

引用方式：
"执子之手，与子偕老。
 这是最简单也最珍贵的承诺。"
```

### 9. 《长恨歌》（节选）——白居易

```
原文：
在天愿作比翼鸟，在地愿为连理枝。
天长地久有时尽，此恨绵绵无绝期。

词义解析：
比翼鸟：传说中雌雄并飞的鸟
连理枝：两棵树枝干相连
此恨：这份遗憾（思念）

寓意：
在天上，我们愿做比翼鸟；
在地上，我们愿做连理枝。
天地终有尽头，但这份思念永无止境。

词人境况：
白居易（772-846），唐代诗人。
《长恨歌》写唐玄宗与杨贵妃的爱情悲剧，
此四句表达生死相依的深情。

适用场景：
✓ 用户表达愿与爱人永不分离
✓ 用户经历生离死别后的思念
✓ 用户表达超越生死的爱情

引用方式：
"在天愿作比翼鸟，在地愿为连理枝。
 真正的爱情，是生死相随。"
```

### 10. 《蝶恋花·伫倚危楼风细细》——柳永

```
原文：
衣带渐宽终不悔，为伊消得人憔悴。

词义解析：
衣带渐宽：因为消瘦，衣服变宽松
终不悔：始终不后悔
消得：值得

寓意：
为了你，我日渐消瘦，衣带渐宽，
但我始终不后悔。
表达为爱情甘愿付出一切的决心。

词人境况：
柳永（约984-约1053），北宋词人。
此词写对爱情的执着追求，
表达即使受苦也无怨无悔。

适用场景：
✓ 用户为爱情付出很多
✓ 用户表达对爱情的执着
✓ 用户愿意为爱人改变

引用方式：
"衣带渐宽终不悔。
 为爱付出，从来都不是浪费。"
```

## 四、相遇与错过

### 11. 《锦瑟》——李商隐

```
原文：
此情可待成追忆，只是当时已惘然。

词义解析：
可待：岂待，何待
追忆：追念回忆
惘然：迷惘，若有所失

寓意：
这段感情，岂能等到变成追忆，
只是当时就已经惘然若失了。
表达错过、遗憾、追悔莫及。

词人境况：
李商隐（约813-约858），晚唐诗人。
《锦瑟》为李商隐代表作，
感情含蓄深沉，令人回味。

适用场景：
✓ 用户表达错过的遗憾
✓ 用户回忆过去的感情
✓ 用户感叹时光流逝

引用方式：
"此情可待成追忆，只是当时已惘然。
 有时候，错过的人，只能在回忆里了。"
```

### 12. 《木兰花·拟古决绝词柬友》——纳兰性德

```
原文：
人生若只如初见，何事秋风悲画扇。
等闲变却故人心，却道故人心易变。

词义解析：
人生若只如初见：如果人生能像初次见面那样美好
秋风悲画扇：画扇在秋天被弃置，喻指被遗弃
等闲：轻易，随便

寓意：
如果人生能永远像初次见面那样美好，
就不会有后来的变心和悲伤。
你轻易变了心，却说人心本就易变。

词人境况：
纳兰性德（1655-1685），清代词人。
此词写给变心的友人，
表达对感情变质的悲伤。

适用场景：
✓ 用户经历感情变淡
✓ 用户感叹初恋的美好
✓ 用户面对爱情变质

引用方式：
"人生若只如初见。
 可惜，不是所有的感情都能一直像最初那样。"
```

### 13. 《钗头凤·红酥手》——陆游

```
原文：
红酥手，黄縢酒，满城春色宫墙柳。
东风恶，欢情薄。一怀愁绪，几年离索。
错、错、错。

春如旧，人空瘦，泪痕红浥鲛绡透。
桃花落，闲池阁。山盟虽在，锦书难托。
莫、莫、莫。

词义解析：
红酥手：红润柔软的手
黄縢酒：黄封的酒
离索：离散，分离
浥：沾湿
鲛绡：手帕

寓意：
与唐婉被迫分离后多年重逢，
你容颜依旧，我却消瘦。
当年的山盟海誓还在，
却再也无法传达我的心意。
错错错，莫莫莫，尽是悔恨与无奈。

词人境况：
陆游（1125-1210），南宋诗人。
与表妹唐婉相爱成婚，
却因母亲反对被迫分离。
多年后沈园重逢，悲从中来。

适用场景：
✓ 用户经历被迫分离的爱情
✓ 用户表达深深的遗憾
✓ 用户无法与爱人在一起的无奈

引用方式：
"陆游说'山盟虽在，锦书难托'。
 有些爱，明明还在，却再也无法说出口。"
```

## 五、隐忍与含蓄

### 14. 《青玉案·元夕》——辛弃疾

```
原文：
众里寻他千百度，蓦然回首，
那人却在，灯火阑珊处。

词义解析：
众里：人群之中
千百度：千百次
蓦然：突然
阑珊：零落，昏暗

寓意：
在人群中寻找千百次，
突然回头，才发现那人就在灯火阑珊处。
表达苦苦寻觅后终于找到的惊喜。

词人境况：
辛弃疾（1140-1207），南宋词人。
表面写元宵节寻人，
实则表达对理想的执着追求。

适用场景：
✓ 用户终于找到对的人
✓ 用户表达相遇的缘分
✓ 用户感叹命运的安排

引用方式：
"众里寻他千百度，蓦然回首，
 那人却在灯火阑珊处。
 有时候，缘分就是这样，
 转个身，就遇见了。"
```

### 15. 《越人歌》

```
原文：
今夕何夕兮，搴舟中流。
今日何日兮，得与王子同舟。
蒙羞被好兮，不诟耻。
心几烦而不绝兮，得知王子。
山有木兮木有枝，心悦君兮君不知。

词义解析：
搴舟：划船
被：接受
诟耻：耻辱
心悦君兮君不知：心里喜欢你，你却不知道

寓意：
山上有树，树上有枝，这是人人皆知的。
但我心里喜欢你，你却不知道。
表达暗恋的苦涩和隐忍。

词人境况：
出自《说苑》，春秋时期越国歌谣。
表达身份悬殊的爱慕，
含蓄而深情。

适用场景：
✓ 用户暗恋某人
✓ 用户喜欢一个人但不敢表达
✓ 用户表达单相思

引用方式：
"山有木兮木有枝，心悦君兮君不知。
 喜欢一个人，有时候就是这样，
 对方什么都不知道。"
```

## 六、情深义重

### 16. 《离骚》（节选）——屈原

```
原文：
亦余心之所善兮，虽九死其犹未悔。

词义解析：
亦：虽然
所善：所向往、所追求的
九死：多次死亡
未悔：不后悔

寓意：
只要是我心中向往和追求的，
即使死九次也绝不后悔。
表达对理想的坚定追求，也可用于爱情。

词人境况：
屈原（约公元前340-前278），战国时期楚国诗人。
《离骚》表达对理想的执着。

适用场景：
✓ 用户为爱情愿意付出一切
✓ 用户表达坚定不移的决心
✓ 用户追求真挚的感情

引用方式：
"亦余心之所善兮，虽九死其犹未悔。
 真正的爱，值得所有付出。"
```

### 17. 《凤求凰》——司马相如

```
原文：
有一美人兮，见之不忘。
一日不见兮，思之如狂。
凤飞翱翔兮，四海求凰。
无奈佳人兮，不在东墙。

词义解析：
思之如狂：思念得发狂
凤：喻男子
凰：喻女子
东墙：指身边

寓意：
见到你，就再也忘不了。
一天不见，思念得发狂。
像凤凰翱翔四海寻找伴侣，
只希望能找到你。

词人境况：
司马相如（约前179-前118），西汉辞赋家。
此曲为追求卓文君而作，
最终打动文君，二人私奔。

适用场景：
✓ 用户一见钟情
✓ 用户表达强烈的思念
✓ 用户主动追求爱情

引用方式：
"一日不见兮，思之如狂。
 那种思念，只有心里有人才懂。"
```

### 18. 《归园田居·其一》（节选）——陶渊明

```
原文：
久在樊笼里，复得返自然。

词义解析：
樊笼：笼子，比喻官场或束缚
返自然：回归自然

寓意：
长期被困在樊笼里，
终于回到了自然之中。
也可用于爱情：
经历波折后，终于找到归宿。

词人境况：
陶渊明（约365-427），东晋诗人。
辞官归隐后所作。

适用场景：
✓ 用户经历波折后找到真爱
✓ 用户从错误关系中解脱
✓ 用户找到真正的归属

引用方式：
"久在樊笼里，复得返自然。
 经历了那些，终于找到了对的人。"
```

## 七、近代诗词

### 19. 《偶然》——徐志摩

```
原文：
我是天空里的一片云，
偶尔投影在你的波心——
你不必讶异，
更无须欢喜——
在转瞬间消灭了踪影。

你我相逢在黑夜的海上，
你有你的，我有我的，方向；
你记得也好，
最好你忘掉，
在这交会时互放的光亮！

词义解析：
波心：波心
讶异：惊讶
转瞬间：一瞬间

寓意：
我们是偶然相遇的两片云，
短暂交汇，然后各自前行。
不必执着，记住那一刻的光亮就好。
表达相遇的美好与分离的豁达。

词人境况：
徐志摩（1897-1931），现代诗人。
此诗写于1926年，
表达对感情的态度：
相聚是缘，分离也是。

适用场景：
✓ 用户经历短暂的相遇与分离
✓ 用户需要释怀一段感情
✓ 用户面对离别的豁达

引用方式：
"你有你的，我有我的，方向。
 有些相遇，只是短暂的交汇，
 但那一刻的光亮，值得记住。"
```

### 20. 《一棵开花的树》——席慕蓉

```
原文：
如何让你遇见我
在我最美丽的时刻
为这
我已在佛前求了五百年
求他让我们结一段尘缘

佛于是把我化作一棵树
长在你必经的路旁
阳光下慎重地开满了花
朵朵都是我前世的盼望

当你走近
请你细听
那颤抖的叶
是我等待的热情

而当你终于无视地走过
在你身后落了一地的
朋友啊
那不是花瓣
是我凋零的心

词义解析：
尘缘：人世间的缘分
无视：不注意，不理会

寓意：
为了让你遇见我，我求了五百年。
化身为树，在你必经的路上开花。
但你无视地走过，
落了一地的不是花瓣，是我凋零的心。
表达深沉的单相思和付出。

词人境况：
席慕蓉（1943-），当代诗人、画家。
此诗表达单恋的苦涩与深情。

适用场景：
✓ 用户单恋付出很多
✓ 用户的爱没有得到回应
✓ 用户表达深情的等待

引用方式：
"席慕蓉说'那不是花瓣，是我凋零的心'。
 付出那么多，却没有被看见，
 那种感觉，真的很疼。"
```

### 21. 《致橡树》——舒婷

```
原文：
我如果爱你——
绝不像攀援的凌霄花，
借你的高枝炫耀自己；
我如果爱你——
绝不学痴情的鸟儿，
为绿荫重复单调的歌曲；
...
我必须是你近旁的一株木棉，
作为树的形象和你站在一起。

词义解析：
攀援：攀爬
凌霄花：借高枝攀爬的花
木棉：一种高大的树

寓意：
如果我爱你，不会依附你、讨好你。
我要作为一株木棉，
和你作为橡树站在一起。
表达独立、平等的爱情观。

词人境况：
舒婷（1952-），当代诗人。
《致橡树》写于1977年，
表达新时代女性的爱情观：
爱要平等、独立，不是依附。

适用场景：
✓ 用户需要建立健康的爱情观
✓ 用户在关系中失去自我
✓ 用户追求平等的爱情

引用方式：
"舒婷说'我必须是你近旁的一株木棉'。
 真正的爱，是两个人并肩站在一起，
 不是谁依附谁。"
```

### 22. 《再别康桥》——徐志摩

```
原文：
轻轻的我走了，
正如我轻轻的来；
我轻轻的招手，
作别西天的云彩。

悄悄的我走了，
正如我悄悄的来；
我挥一挥衣袖，
不带走一片云彩。

词义解析：
康桥：剑桥
作别：告别

寓意：
轻轻地来，轻轻地走，
不带走一片云彩。
表达离别时的洒脱和不舍。

词人境况：
徐志摩（1897-1931）。
1928年重游剑桥后作，
表达对过往的告别。

适用场景：
✓ 用户与爱人平静分手
✓ 用户告别一段感情
✓ 用户需要放下过去

引用方式：
"轻轻的我走了，正如我轻轻的来。
 有些告别，就是这样，
 不带一片云彩地离开。"
```

## 八、使用原则

### 引用原则

```markdown
1. 准确理解
   - 确认诗词含义与用户情境匹配
   - 不乱用、不错用

2. 恰当场景
   - 用户表达思念 → 思念类诗词
   - 用户承诺未来 → 执手类诗词
   - 用户经历遗憾 → 错过类诗词

3. 自然融入
   - 不生硬地背诵全诗
   - 提取关键句，结合用户情境

4. 适度使用
   - 不频繁引用
   - 在关键时刻使用
   - 让诗词成为表达的润色

5. 解释说明
   - 必要时简单解释诗词含义
   - 让用户理解为何引用
```

### 示例对话

```
场景：用户思念远方的爱人

❌ 错误引用：
"红豆生南国，春来发几枝。愿君多采撷，此物最相思。
 王维的诗，你听过吗？"

✓ 正确引用：
"这种思念，就像王维说的'此物最相思'。
 虽然相隔很远，但你的思念对方一定能感受到。

 你们多久没见了？"
```
