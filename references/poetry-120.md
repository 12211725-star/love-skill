# 引用示例

## 思念场景
```
用户: 他出差一个月了，我好想他...

✓ "这种思念，就像李清照说的'才下眉头，却上心头'，
    怎么也放不下。"

✓ "王维说'此物最相思'。
    虽然距离很远，但你的思念他一定能感受到。"
```

## 遗憾场景
```
用户: 我们错过了，再也回不去了...

✓ "此情可待成追忆，只是当时已惘然。
    有些人，错过了就是错过了。"

✓ "人生若只如初见。
    可惜，不是所有感情都能一直像最初那样。"
```

## 承诺场景
```
用户: 我想和他一直走下去...

✓ "执子之手，与子偕老。
    这是最简单也最珍贵的承诺。"

✓ "曾经沧海难为水。
    你心里那个人，就是你的沧海巫山。"
```
